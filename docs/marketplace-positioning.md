# Marketplace Positioning

## One-line Summary

Claude Code project commands for AI-native web production across webpages, websites, and web services.

## Short Description

이 플러그인은 브리프 작성부터 구조 설계, Figma MCP handoff, 프론트 구현 계획, 검증 루프까지 이어지는 웹 제작 워크플로우를 Claude Code 안에서 재사용 가능한 명령 세트로 제공합니다.

## Target Categories

- Design to code
- Website generation
- Frontend workflow
- AI production pipeline
- Figma MCP workflow

## Key Differentiators

- 홈페이지 전용이 아니라 webpage, website, webservice 전반을 다룸
- Figma MCP handoff와 frontend build spec을 하나의 흐름으로 연결
- 기존 코드베이스가 있을 때와 없을 때의 동작을 분리
- 실험과 검증 루프까지 포함한 운영형 워크플로우

## Suggested Tagline Options

- AI-native web production workflow for Claude Code
- Build webpages, websites, and web services with reusable Claude commands
- From brief to Figma handoff to frontend build, in one Claude workflow

## Suggested Listing Copy

이 플러그인은 Claude Code를 웹 제작 오케스트레이션 도구로 바꾸기 위한 프로젝트 명령 모음입니다. 랜딩 페이지, 웹사이트 리뉴얼, SaaS 웹서비스 기획과 구현 계획까지 하나의 파이프라인으로 연결할 수 있게 설계했습니다.
